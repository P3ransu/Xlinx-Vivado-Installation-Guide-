# Xlinx-Vivado-Installation-Guide-
---
## This is a quick, simple and easy guide on how to install Xlinx Vivado for the purpose of Design,Simulation,Synthesis and Implementation of HDL programs.
---
## Some Pre Checks To Confirm before starting
1. Make sure you have a stable connection
2. Make sure to use Either Microsoft Edge or Google Chrome to avoid any unforseen issues
3. maintain a empty disk space of about 80GB -200GB

## Installation Process
---
### Step 1
1. First got to the download webpage of Vivado on the amd website (https://www.amd.com/en/products/software/adaptive-socs-and-fpgas/vivado.html)
proceed to click the downaload button which would take you to the download page
2. Here as per system requirements please click the text on the left and not the unified System Installer 
This will take you to the information page
3. eneter your information as required and make sure to either use a college email or the company email
and proceed 
4. in case of the license not being issued please do proceed to send an email for which a reply will be given within 3 days
5. now considering this is done click the downnload button and choose the file destination for the installer
now the installation may take some time
6. upon completion run the isntaller and follow the steps as given in the wizard that opens
7. proceed to next
8. enter email n pass of created account
9. select the required product for installation
10. select the required devices 
11. please note the storage taken will be massive so refrain from checking unnesacery options
proceed to agree with all the license agreements
select the folder to be installed in 
click next 
click install
